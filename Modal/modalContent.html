This is a Modal!

This is a modal for the Four51 OrderCloud 2.0 Storefront It's made with Angular JS and UI-Bootstrap. You are looking at the html found in the modalContent.html file. The link to this demo of the modal was placed in 'partials/branding.html', but you can place it virtually anywhere.

Here is a list of uses:

  1. tell users pertinent information
  2. alert users that they're leaving the site and are going to a different link
  3. display info without going to a different page (contact information, small 'FAQs', etc.)
  4. anything you need to get a message across conveniently
